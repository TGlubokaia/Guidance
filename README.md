## Неделя вожатства

Участие в первом этапе командной разработки проекта "Неделя вожатства".

С моей стороны были реализованы:
- форма для данных участника и валидация полей
- форма для добавления мероприятия и валидация полей


Запустить проект:

```bash
npm run dev
```
